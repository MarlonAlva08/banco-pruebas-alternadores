Banco de Pruebas de Alternadores

Proyecto de ingeniería para verificar y analizar alternadores de forma precisa y segura.

Características del proyecto

  • ✅ Medición de voltaje, corriente y frecuencia
  • ✅ Prueba bajo carga simulada
  • ✅ Interfaz con Arduino y sensores
  • ✅ Diseño portátil con protección eléctrica

Este banco fue diseñado para talleres eléctricos, estudiantes y docentes que desean verificar alternadores de forma segura y confiable.
